# 🌊 KahootFlow (Status WIP)

A Telegram bot client for Kahoot.

## Stack:

- **[@telegraf-hardened/fetch](https://github.com/telegraf-hardened/fetch)**: A robust `undici`-based fetch implementation with native SOCKS5/HTTP proxy support.
- **[telegraf-hardened](https://github.com/telegraf-hardened/telegraf)**: A strict, security-focused fork of Telegraf.js.
